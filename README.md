# üé® MoodBoarder

*Translate Your Mood into Visual Inspiration.* MoodBoarder is a elegant, single-page web application that allows you to search for images based on emotions and aesthetics rather than concrete keywords. Powered by the Unsplash API and built with a modern glassmorphism design.

## ‚ú® Features

- *Emotion-Based Search:* Find images by feeling (e.g., "serene," "melancholy," "energetic").
- *Stunning UI:* Features a modern dark theme with animated particle background and glassmorphism effects.
- *Responsive Design:* Fully responsive layout that works seamlessly on desktop, tablet, and mobile.
- *Masonry Grid:* Beautiful Pinterest-style image grid for an optimal visual experience.
- *Loading States:* Elegant loading spinner and user-friendly error handling.

## üì¶ Installation & Local Setup

Want to run this project locally? Follow these steps:

1.  *Clone the repository:*
    bash
    git clone https://github.com/your-username/moodboarder.git
    cd moodboarder
    

2.  *Get your Unsplash API Key:*
    - Visit [Unsplash Developers](https://unsplash.com/developers)
    - Register a new application (it's free).
    - Copy your Access Key.

3.  *Add your API Key:*
    - Open the script.js file.
    - Find the const ACCESS_KEY variable.
    - Replace 'YOUR_UNSPLASH_ACCESS_KEY_HERE' with your actual key.
    javascript
    const ACCESS_KEY = 'your_actual_key_here'; // Without quotes around the key itself
    

4.  *Open it in your browser:*
    - Simply open the index.html file in a web browser.
    - Or, for a better experience, use a local server. With Python installed:
      bash
      # Python 3.x
      python -m http.server 8000
      
      Then visit http://localhost:8000 in your browser.
